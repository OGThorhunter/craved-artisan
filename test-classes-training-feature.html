Classes/Training Feature Implementation Test

Testing the new "Classes / Training" product category with available seats, cost per seat, and duration functionality.

Feature Overview

The Classes/Training feature allows vendors to create educational products like cooking classes, workshops, and seminars with the following unique characteristics:

  • Available Seats: Number of participants the class can accommodate
  • Cost per Seat: Price each participant pays to attend
  • Duration: Length of the class (e.g., "3 hours", "2 days")
  • Total Revenue: Automatically calculated as Available Seats √ó Cost per Seat

How to Test in the Application

  1. Navigate to Products Page: Go to the vendor products dashboard
  2. Create New Product: Click "Add Product" button
  3. Select Classes Type: Choose "Classes / Training" from the Type dropdown
  4. Fill Class Details:
    • Enter product name (e.g., "Sourdough Making Class")
    • Add description explaining the class content
    • Set available seats (e.g., 25)
    • Enter cost per seat (e.g., $75.00)
    • Specify duration (e.g., "3 hours")
  5. Verify Auto-calculation: Check that total price = seats √ó cost per seat
  6. Save Product: Complete the form and save
  7. View Product Card: Verify the purple CLASSES badge and class details
  8. Test View Modal: Click "View" to see detailed class information
  9. Test Edit Modal: Click "Edit" to modify class details
